# Landing Page v2 Spec

## Amaç
Public ziyaretçiye 5 saniyede şunları anlatmak:
1. VAULT doğrulanmış luxury emlak profesyonelleri içindir.
2. WhatsApp grup kaosunu çözer.
3. Kapalı portföyleri harita, AI eşleşme, detay talebi ve Share Studio ile yönetir.
4. Private beta/davetli network'tür.

## Kritik görsel gereksinimler
- Hero sağında tek ana map-first ürün paneli + 2–3 floating card.
- Map gece modu/dark satellite/gold pins olmalı.
- VAULT Asistan chat mockup gerçek ürün gibi görünmeli.
- Profesyonel Ağ kartlarında avatar/name overlap sistemsel çözülmeli.
- Share Studio screenshot'taki 3-column workstation yapısına yaklaşmalı.
- Kapalı portföy detail section eklenmeli: hero image, teaser, kilitli bilgiler, detail request.
