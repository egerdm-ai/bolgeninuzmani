# User Flows

## Flow A — Portföy ekle ve paylaş
1. Portföylerim → Portföy Oluştur.
2. AI ile veya manuel portföy taslağı.
3. Konum/fiyat/kategori detayları.
4. Fotoğraf/PDF/belge ekle.
5. Gizlilik: tam adres, telefon, PDF kilitli.
6. Yayınla.
7. Sistem eşleşen arayışları gösterir.
8. Share Studio ile WhatsApp/link/PDF paylaş.

## Flow B — Müşteri arayışı kaydet
1. Arayışlarım → Yeni Arayış Oluştur.
2. Doğal dil: “Bebek’te havuzlu 5+1 villa...”
3. VAULT Asistan filtrelere çevirir.
4. Kullanıcı kriterleri düzenler.
5. Eşleşmeleri bul ve kaydet.
6. Yeni portföy gelince bildirim al.

## Flow C — Network arayışından fırsat bul
1. Keşfet → Arayışlar.
2. Filtre: Portföylerimle eşleşenler.
3. Arayış detayına gir.
4. Benim uygun portföylerimden birini öner.
5. Profesyonelle detay talebi/iletişim başlat.

## Flow D — Portföy arama ve detay talebi
1. Keşfet → Portföyler.
2. Harita/filtre/AI search ile portföy bul.
3. Portföy detayına gir.
4. Kilitli bilgileri gör.
5. Detay Talebi Gönder.
6. Onay gelirse full PDF/adres/telefon açılır.

## Flow E — Bölge takip et
1. Keşfet → Bölgeler.
2. Bölge detayı: Yalıkavak/Bebek/Riva.
3. Bölgeyi Takip Et.
4. Yeni portföy/arayış/uzman aktivitesi bildirim olur.

## Flow F — VAULT Asistan
1. VAULT Asistan’a sor.
2. Kullanıcı portföy arar, arayış oluşturur, match analizi ister veya PDF hazırlar.
3. AI structured sonuç üretir.
4. Sonuçlardan ilgili route'lara gider: portföy, arayış, profil, share studio.
