# Design System & Mobile Strategy

## Visual direction
- Dark luxury: black/charcoal base.
- Gold accents selective.
- Glassy cards, subtle borders, soft glow.
- Serif display typography for premium headlines.
- Dense dashboard layouts but readable.

## UI foundations
- AppShell sidebar + topbar.
- Keşfet grouped navigation.
- Cards: glass, subtle border, hover gold.
- Badges: verified, elite/pro/private beta, region expert, locked, match score.
- Modals/drawers for filters, detail request, match explanation.
- Sticky tab bars for profile/detail pages.

## Mobile priorities
- Dashboard mobile later but responsive baseline required.
- Landing excellent mobile.
- Search: map/list toggle, filter drawer full-screen.
- Portfolio detail: image gallery first, sticky detail request CTA.
- Professional profile: tabs horizontal scroll, catalog list.
- Share Studio: channel cards stacked.

## Known visual bugs to solve
- Professional card/profile hero avatar-name-cover overlap: component-level fix required.
- Hero landing map must be dark/night/gold not generic.
- Share Studio landing should use stronger 3-column product mockup.
