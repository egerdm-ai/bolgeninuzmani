# Feature Scope — MVP / P1 / P2

## MVP
- Auth + profile + verification status.
- Portföy CRUD: create draft, publish, edit, list, detail.
- Media upload: photos, cover, documents basic.
- Privacy/access model: locked fields, detail request, approve/reject.
- Map-first portfolio search + Airbnb-style filters.
- Profesyoneller + profile page + follow local/persistent.
- Bölgeler + region detail + region watch basic.
- Arayışlarım: create/edit/save/search notifications config.
- Arayışlar: network buyer search discovery.
- Eşleşmeler: basic match score + explanation.
- Share Studio: WhatsApp message, link preview, basic teaser PDF.
- VAULT Asistan V1: query → filters, AI import text/PDF mock to real later.
- Notifications in-app.
- Landing + public `/p` and `/v` preview.

## P1
- Real AI Import from WhatsApp text/PDF/images.
- Semantic search with embeddings/pgvector.
- Better matching ranking and scheduled jobs.
- Real PDF generation with templates.
- Share analytics.
- Duplicate portfolio detection.
- Region analytics + region expert scoring.
- Team/company profiles.
- Email/WhatsApp notification delivery.

## P2
- WhatsApp Business Cloud API integration.
- Sahibinden/Facebook URL import where allowed.
- CRM/opportunity pipeline.
- Office admin and team performance.
- Advanced valuation model and market data integrations.
- Mobile app.
