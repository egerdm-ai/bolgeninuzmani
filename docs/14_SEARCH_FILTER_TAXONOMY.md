# Search Filter & Property Taxonomy

## Ana kategori
- Konut
- Villa / Yalı
- Arsa / Arazi
- Ticari
- Otel / Turizm
- Bina
- Endüstriyel
- İşletme / Devren
- Özel Varlık

## Ortak filtreler
- Şehir, ilçe, mahalle, bölge
- Haritada alan seç
- Portföy tipi, alt tip
- Satılık/kiralık/sezonluk/devren
- Fiyat aralığı, para birimi, m² fiyatı
- Brüt/net/arsa/kapalı/açık alan
- Oda, salon, yatak odası, banyo, WC, otopark kapasitesi
- Fotoğraflı, PDF hazır, video, harita
- Detay talebi gerekli, tam adres gizli, telefon gizli
- Bölge uzmanından, takip ettiğim profesyoneller, verified
- Arayışlarımla eşleşenler, veri skoru yüksek, AI önerili

## Konut/Villa detayları
- Bina yaşı, kat, kat sayısı
- Isıtma, mutfak tipi, balkon, teras, bahçe
- Site içinde, aidat, eşyalı, kullanım durumu
- Tapu durumu, krediye uygunluk, vatandaşlık uygunluğu
- Deniz/boğaz/orman/şehir manzarası
- Havuz, kapalı havuz, sonsuzluk havuzu
- Akıllı ev, güvenlik, jeneratör, personel odası, misafir evi
- İskele, denize sıfır, helipad, sinema, spa, sauna, hamam

## Arsa detayları
- İmar durumu
- Ada/parsel/pafta
- KAKS/emsal, TAKS, gabari, Hmax
- Yol cephesi, parsel derinliği, ana yola/denize uzaklık
- Tek tapu, hisseli, ifrazlı, kat karşılığı uygun
- Elektrik, su, kanalizasyon, doğalgaz, yol, trafo
- Villa imarlı, turizm imarlı, ticari imarlı, sanayi imarlı

## Ticari/Endüstriyel detayları
- Ticari tip: ofis, dükkan, restoran, depo, fabrika, otel
- Kapalı/açık alan, cephe genişliği, tavan yüksekliği
- Ruhsat, kiracı, kira getirisi, devren bedeli, ciro bilgisi
- Jeneratör, yangın sistemi, yükleme alanı, tır girişi, OSB

## UI prensibi
Filtre modalı Airbnb gibi kolay, Sahibinden gibi detaylı, VAULT gibi gizlilik/eşleşme odaklı olmalı.
